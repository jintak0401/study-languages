# 2026-05-31 · て형 (te-form)

오늘의 핵심: **て형**. 과거 た형을 알면 거저먹기 — **た·だ 자리에 て·で만** 넣으면 끝.

## 규칙 요약
- 2그룹: る → **て** (食べる→食べて)
- 1그룹: う·つ·る→**って** / む·ぶ·ぬ→**んで** / く→**いて** / ぐ→**いで** / す→**して**
- 불규칙: する→**して**, 来る→**来て**
- ⚠️ 예외: 行く→**行って**

## て형으로 할 수 있는 것
- **〜てください** — 요청: 窓を開けてください (창문 열어 주세요)
- **〜ています** — 진행·상태: 食べています (먹고 있다) ← 전에 헷갈렸던 「見てます」의 정체!
- **〜てもいいですか** — 허가: 写真を撮ってもいいですか (사진 찍어도 될까요?)

## 새 단어
書く(쓰다)·聞く(듣다/묻다)·遊ぶ(놀다)·泳ぐ(헤엄치다)·持つ(들다) — 각각 て형 그룹 연습용.

## 미니 과제
다음을 て형으로: 飲む / 行く / 話す / 見る / する → 정답은 Grammar 카드의 "자세히"에서 표로 확인.
